Let's get started!

Apple Watch

Duo to the broad usage of the Apple Watch, it's been choosen as the IMU to track certain events. This Bachelor Thesis provides a framework to inspect sensor data in near real-time.

Logging

Using a combination of InfluxDB, Telegraf and Grafana you can log various sensor data events from the Apple Watch in real-time.

Sit-Stand Detector

Use the SitStand Watch App to detect wether a paticipant stood up or sat down. Every event get's logged in a dashboard so that you can monitor your participants.
